   - 专门为 Tongbu.Tui.Nms.Inner 项目定制
   - 已配置好所有参数（Token、Project ID、Base URL）
   - 直接运行即可获取提交
   - 输出 JSON 格式 + 可读格式摘要

3. fetch_with_config.py  🔒 安全推荐
   - 使用配置文件的版本
   - Token 不暴露在代码中
   - 更适合团队协作和版本控制

==================================================================================
  快速使用指南
==================================================================================

【方式 1】最简单 - 直接运行
--------------------------
python fetch_tongbu_commits.py

输出：JSON 格式的提交数据 + 可读摘要


【方式 2】指定参数
------------------
# 获取 dev 分支最新 20 条
python fetch_tongbu_commits.py --branch dev --per-page 20

# 保存到文件
python fetch_tongbu_commits.py --output commits.json

# 只输出 JSON
python fetch_tongbu_commits.py --json-only


【方式 3】使用配置文件（更安全）
--------------------------------
# 首次：复制配置文件
copy config_example.json config.json

# 运行
python fetch_with_config.py


【方式 4】Python 代码中调用
---------------------------
from fetch_tongbu_commits import fetch_tongbu_commits
import json

result = fetch_tongbu_commits(branch='dev', per_page=20)
print(json.dumps(result, indent=2, ensure_ascii=False))


【测试功能】
-----------
python test_tongbu.py
